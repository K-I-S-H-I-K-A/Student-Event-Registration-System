# Student-Event-Registration-System!
**Course:** SODV1201 - Introduction to Web Programming  

##  About the Project
This app is a group project designed to connect property owners with people looking for coworking spaces (desks, meeting rooms, etc.). It works similarly to Airbnb, allowing owners to list their space and coworkers to search and book.

##  Phase 1 
- Created the Landing Page, Header, and Footer.
- Built forms to handle User Signup (Owner/Coworker roles) and login.
- Created functional pages for Owners to add/list their properties and workspaces.
- Added a search and filter feature for Coworkers to find spaces.
- Added a booking page, booking confirmation and profile page.
- **NOte** At this stage, all data is stored in Json.
