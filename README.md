# Student Event Registration System

**Course:** SODV1201 - Introduction to Web Programming  

--

## About the Project
This application is a group project designed to connect property owners with people looking for coworking spaces (desks, meeting rooms, etc.). It works similarly to Airbnb, where owners can list their spaces and coworkers can search and book them.

--

## Phase 1
- Created landing page, header, and footer
- Built signup and login forms (Owner / Coworker roles)
- Added functionality for owners to create and manage properties and workspaces
- Implemented search and filter features for coworkers
- Added booking page, booking confirmation, and profile page
- All data is stored using JSON (in-memory storage)

--

## Phase 2
In Phase 2, the project is converted into a full-stack web application by adding a backend server and database.

--

## Backend Features
- Built using Node.js and Express
- REST API for main functionality
- Connected to a database for permanent data storage
- User authentication (signup and login)
- Role-based access control (Owner and Coworker)
